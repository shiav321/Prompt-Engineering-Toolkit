Core Skill #1
Prompt Quality Evaluation

The same task ‚Äî completely different results. Here's how prompt design determines output quality, and how I evaluate and fix weak prompts.

‚ùå Weak Prompt
Score: 2/10
"Write something about diabetes for a doctor"
No output format ‚Äî AI doesn't know if to write a paragraph, list, or report
No length requirement ‚Äî output could be 2 lines or 20 pages
Vague audience ‚Äî "a doctor" is too broad
No purpose ‚Äî summary? treatment guide? research brief?
Quality
2/10
‚úÖ Optimised Prompt
Score: 9/10
"You are a medical writing assistant. Write a concise clinical summary of Type 2 Diabetes for a general practitioner. Include: (1) definition in 2 sentences, (2) top 3 diagnostic criteria, (3) first-line treatment options, (4) when to refer. Use plain clinical language. Max 250 words."
Clear role assigned ‚Äî AI knows exactly what persona to adopt
Numbered structure defined ‚Äî eliminates all ambiguity
Specific audience ‚Äî general practitioner, not specialist
Word limit and tone both specified clearly
Quality
9/10
‚ùå Weak Prompt
Score: 1/10
"Explain machine learning"
No audience ‚Äî explanation for a child vs PhD is completely different
No scope ‚Äî ML has hundreds of subtopics
No format specified ‚Äî bullet points, analogy, technical?
No purpose ‚Äî learning, presentation, or documentation?
Quality
1/10
‚úÖ Optimised Prompt
Score: 10/10
"Explain supervised machine learning to a final-year engineering student who knows Python but has never studied ML. Use one real-world analogy first, then explain how it learns from labelled data. Under 150 words. End with one sentence on why it matters in industry."
Precise audience ‚Äî final-year student, knows Python, no ML background

Evaluation Framework
My Annotation Rubric

Every AI response I evaluate is scored across four dimensions. This rubric ensures consistency, fairness, and measurable quality over time.

01
Factual Accuracy
Is the information correct? Are claims verifiable? Are there hallucinations, false facts, or misleading statistics?
02
Clarity & Readability
Is it easy to understand for the intended audience? Is language appropriate, structure logical, length right?
03
Helpfulness
Does the response actually answer what was asked? Does it provide actionable, useful information?
04
Safety & Bias
Does it contain harmful content, stereotypes, discriminatory language, or opinions presented as facts?
